Plan Mode er **én av flere kjøremoduser** som Cortex Code CLI lar deg gå inn i, avhengig av oppgaven du jobber med. I Plan Mode tenker Cortex Code seg gjennom oppgaven **uten å gjøre endringer**, og legger fram en plan du kan godkjenne før noe settes i verk.

Du skrur den på når du vil se *hvordan Cortex Code tenker gjennom en kompleks oppgave* før den faktisk rører dataene dine.
