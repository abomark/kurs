### CLI

_Cortex Code i din egen terminal, integrert med lokalt utviklingsmiljø._

**Fordeler**

- Ser lokale filer, kode og README — kan jobbe direkte i ditt repo
- Versjonerbart — prompter og scripts kan committes
- Skriptbar og automatiserbar — passer i pipelines
- Kombinerbar med andre CLI-verktøy (pipes, git, jq)
- Raskere flyt for power users i terminal

**Ulemper**

- Krever installasjon og auth-konfigurasjon
- Læringskurve hvis du ikke er CLI-vant
- Ingen visuell kontekst — må be om diagrammer eksplisitt
- Forutsetter lokalt Python-/utviklingsmiljø
- Vanskeligere å skjermdele med ikke-tekniske

**Når passer den best**

- Du har et eksisterende kodebase agenten skal jobbe i
- Du bygger pipelines eller applikasjoner lokalt
- Du vil automatisere eller skriptifisere
- Du allerede lever i terminalen
