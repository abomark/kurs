### Snowsight

_Cortex Code i nettleseren via Snowflakes web-UI._

**Fordeler**

- Ingen installasjon — bare logg inn i nettleseren
- Visuell kontekst — ser tabeller, resultater og dashbord inline
- RBAC og auth er allerede satt opp gjennom Snowflake-økten
- Enkel å skjermdele med ikke-tekniske kolleger
- Bra for utforsking og ad-hoc-arbeid

**Ulemper**

- Vet ingenting om dine lokale filer eller repo
- Vanskelig å versjonere eller dele prompter via git
- Begrenset til nettleser-context — kan ikke lese kodebase
- Tregere flyt for repetitive oppgaver
- Vanskelig å integrere i CI/CD eller automatisering

**Når passer den best**

- Ad-hoc data exploration
- Demo og opplæring
- Skjermdeling med kolleger
- Du er ny til Cortex Code og vil teste raskt
