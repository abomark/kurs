1. Åpne Cortex Code i Snowsight.
2. Formuler en bevisst kompleks oppgave, f.eks.:
   > _Bygg en avledet tabell som aggregerer X per kunde og legg på en
   > kvalitetssjekk._ (noe som krever flere steg)
3. Skru på Plan Mode med `Ctrl + P` eller `/plan`. Agenten går read-only.
4. Les den strukturerte flertrinnsplanen agenten legger fram. Legg merke til:
   - Hvilke steg den deler oppgaven inn i
   - Hvilke antagelser den gjør
   - Hvilke tabeller/objekter den ville rørt
5. Godkjenn, juster eller avvis planen — *du* bestemmer før noe utføres.
