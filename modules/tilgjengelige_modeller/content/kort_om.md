En LLM, eller Large Language Model, er en språkmodell som forutsier neste token basert på konteksten den har fått. I praksis er det modellen som «tenker» når agenten løser oppgaver.

Cortex Code er mer enn bare en LLM: det er modellen kombinert med verktøy, kontekst og en arbeidsloop.

De vanligste modellvalgene er:

| Modell     | Brukes når                                                                         |
| ---------- | ---------------------------------------------------------------------------------- |
| **Sonnet** | Standardvalg for det meste: SQL, analyse, forklaring, feilsøking og rask iterasjon |
| **Opus**   | Mer komplekse, tvetydige eller risikofylte oppgaver som krever dypere resonnement  |
| **Haiku**  | Enkle høy-volum-oppgaver der fart og lav kostnad er viktigst                        |
