# Skills i Cortex Code

En **skill** er en gjenbrukbar arbeidsflyt som forteller Cortex Code hvordan den skal håndtere en bestemt Snowflake-oppgave. I stedet for å svare helt åpent og uforutsigbart, gir en skill modellen et fast rammeverk å jobbe innenfor - fra domenekontekst til strukturert output.

Du kaller en skill med samme `@` som du lærte for katalog-referanser - for
eksempel `@(serverSkill:lineage)`.
