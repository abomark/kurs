Cortex Code oppdager skills fra flere steder, avhengig av hvor bredt de skal gjelde:

| Type | Plassering | Omfang |
| --- | --- | --- |
| **Bundled** | Innebygd i Cortex Code | Tilgjengelig som standard |
| **User-level** | `~/.snowflake/cortex/skills/` eller `~/.cortex/skills/` | Tilgjengelig på tvers av prosjekter |
| **Project-level** | `.cortex/skills/` i repoet | Kun tilgjengelig i det prosjektet |
